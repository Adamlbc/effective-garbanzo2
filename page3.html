2. ATI Radeon 7200

The ATI Radeon 7200 is a graphics card that was released in 2000 and is based on the Rage 6 architecture. It features 30 million transistors, a 128-bit memory interface, and supports DirectX 7. The card is powered by 32 MB of DDR memory clocked at 143 MHz, with a memory bandwidth of 5.31 GB. It is a single-slot card that does not require an additional power connector and has a power consumption of 23 Watts.

4. ATI radeon SDR

The ATI Radeon SDR is a desktop graphics card that was released on June 1, 2000. It is based on the Rage 6 microarchitecture and is designed for desktop computers. It has a maximum RAM amount of 32 MB and a memory bus width of 128 Bit. The memory clock speed is 166 MHz, with a memory bandwidth of 2.656 GB. The power consumption is rated at 23 Watts.
